# Docker Compose

Docker Compose is the tool which is used to manage multiple containers.

Docker compose is a tool that is used to define and manage multiple containers.

```Docker compose एक tool है जो multiple docker containers को एक yaml file के through define (yaml file मैं containers की configuration लिखना) करता है और उनको manage करता है, manage करने का मतलब है की सभी containers को एक ही साथ run करना, stop करना, delete करना ये सब|```

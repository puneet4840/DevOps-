# Connect your local system with GitLab to push code

GitLab par project ke ander local code push karne ke liye vaise to do tariko se hota hai: 1 - HTTPS, 2 - SSH.

HTTPS se code push karne par apko login credentails se login karke karna hota hai, lekin SSH se apko system ki public key GitLab mein add karni hoti hai. SSH recommended way hai GitLab pe code push karne ka.

To hum dekhenge ki SSH ke though GitLab par code push kaise karte hain.

<br>
<br>

### Push local code to GitLab using SSH

To hum dekhenge ki local system se GitLab par SSH ke through code ko kaise push karte hain:

**Step-1: Generate Key-Pair in Local System**:

Sabse pehle apne local system par ek ssh key-pari generate karna hota hai.

Open cmd in windows and type below command, For Linux simply type below command.

Command:
```
sss-keygen
```

Ye command apke system par ek public private key-pair generate kar degi.

Jiski location hogi:
- Windows: ```C:\Users\puneetverma02\.ssh```.
- Linux: ```/home/puneet/.ssh```.

<br>

**Step-2: Copy the public key**:

Ab jaha key-pair generate hua tha us location par jakar apko public key copy karni hai. Linux mein cat command se print karke copy kar sakte ho lekin windows mein thoda alag hai.

Windows mein aap pehle is location (```C:\Users\puneetverma02\.ssh```) par jao. Yaha jake cmd open karke VS Code open karo, cmd mein ```. code``` command type karke.

Fir VS Code mein Git Bash select karke aap .pub file ko cat command se open kar sakte ho.

<br>

**Step-3: Paste the SSH key in GitLab**:

Gitlab mein user settings mein jao, apne profile pe click karke preferences pe click karo to aap user setting mein chale jaoge. Waha apko SSH Keys karke option dikhega, uspe click karo uske ander jao.

Ab apko add new key karke public key paste kar deni hai aur save kar deni hai. Isse apke local system ki SSH key GitLab mein add ho jayegi.

<br>

**Step-4: Push Code to GitLab**:

GitLab mein code push karne ke liye apko project ki SSH url git remote mein add karni hogi. Git remote mein url add karne ke baad jab aap git push karoge to code directly usi project ki kisi branch mein push ho jayega.

# Deploying pre-built application on kubernetes using helm

Is slide mein pre-built application kubernetes par deploy karenge helm ke through.

<br>
<br>

### LAB PROJECT: Redis Deployment using Helm Repo

Application: Redis

Helm Chart: Bitnami Redis Helm Chart

<br>
<br>

### WHY REDIS?

Real-world use:
- caching
- session storage
- queues

DevOps perspective:
- Stateful app
- Persistence
- Password (secret handling)
- Scaling

<br>
<br>

### FINAL ARCHITECTURE
```
Client → Redis Service → Redis Pod (Master + Replica)
```

<br>
<br>

### STEP 1: Helm repo add

```
helm repo add bitnami https://charts.bitnami.com/bitnami
```

<br>
<br>

### STEP 2: Repo update
```
helm repo update
```

<br>
<br>

### STEP 3: Redis chart search
```
helm search repo redis
```

Tumhe milega:
```
bitnami/redis
```

<br>
<br>

### STEP 4: Default values samjho
```
helm show values bitnami/redis
```

<br>
<br>

### STEP 5: Custom values.yaml banao

```
architecture: replication

auth:
  enabled: true
  password: "redis123"

master:
  persistence:
    enabled: true
    size: 1Gi

replica:
  replicaCount: 1
```

Explanation

- architecture: replication
- master + replica create karega

auth:
- password protected Redis

persistence:
- data restart ke baad bhi rahega

<br>
<br>

### STEP 6: Install Redis
```
helm install my-redis bitnami/redis -f values.yaml
```

<br>
<br>

### STEP 7: Verify
```
kubectl get pods
```
```
kubectl get svc
```

<br>

Tum kya dekhoge?

Pods:
- redis-master
- redis-replica

Service:
- ClusterIP

<br>
<br>

### STEP 8: Redis access

Redis HTTP app nahi hai.
- CLI se access karte hain

Pod ke andar jao
```
kubectl exec -it my-redis-master-0 -- bash
```

Redis CLI
```
redis-cli -a redis123
```

Test karo
```
SET name Puneet
GET name
```

Output:
```
Puneet
```

<br>
<br>

### UNDER THE HOOD

Flow:
```
Client → Service → Redis Master → Replica sync
```

<br>
<br>

### STEP 9: Scale replica
```
replica:
  replicaCount: 2
```

```
helm upgrade my-redis bitnami/redis -f values.yaml
```

Kya hoga?
- New replica pod create hoga.
- Master se sync karega.

<br>
<br>

### STEP 10: Rollback

```
helm rollback my-redis 1
```

DONE!!!

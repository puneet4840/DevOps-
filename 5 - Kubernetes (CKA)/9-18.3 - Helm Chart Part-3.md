# Helm Chart Pre-built application installation concept

Isse pehle wali slide mein humne dekha ki hum khud se application ko helm se chart se deploy karne ki baat kar rahe the. Lekin is slide mein hum dekhenge ki kaise helm chart se kuch pre-built application kubernetes mein kaise deploy karte hain.

<br>

### Helm se predefined apps ka concept kya hai?

Helm sirf tera custom chart nahi, balki ready-made charts (pre-built apps) bhi provide karta hai.

Inhe bolte hain: **Helm Charts from Repositories**

Jaise:
- Database.
- Monitoring tools
- Logging stack.
- CI/CD tools.

To isse pehle hum khud ke application ko helm chart ke through deploy kar rahe the, lekin hum helm chart ke through pre-built application jaise monitoring tools, logging stack, CI/CD tools aur database bhi install kar sakte hain.

Helm ka sabse bada fayda hi yahi hai ki ye Kubernetes ke liye ek App Store ki tarah kaam karta hai. Isme "Pre-built application" ka matlab hota hai Helm Charts, jo pehle se kisi expert ne likh kar public rakhe hue hain.

Jaise tum Android phone mein Play Store se WhatsApp ya Instagram install karte ho, waise hi tum Helm se Kubernetes mein pre-built apps (jaise MySQL, Nginx, Jenkins) install karte ho.
